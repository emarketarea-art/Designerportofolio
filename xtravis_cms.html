ℹ️ Comment créer un Token GitHub

  1. github.com → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Cliquer Generate new token (classic)
  4. Cocher repo (accès complet au dépôt)
  5. Copier le token généré et le coller ci-dessus
⚠️ Ne partagez jamais ce token. Il est stocké uniquement dans votre navigateur (localStorage) et n'est jamais envoyé ailleurs que vers l'API GitHub.
